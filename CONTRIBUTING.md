# Contributing to Robocopy+

First off, thank you for considering contributing to **Robocopy+**!
We welcome any and all contributions, issues, and PRs. 

## Getting Started

1. **Fork the Repository**: Start by forking `dcryptoniun/robocpyplusgui`.
2. **Clone Locally**: Clone your fork onto your Windows development machine.
3. **Environment setup**: Make sure you have the Rust `stable` toolchain installed via `rustup`. 
4. **Compile**: This project uses `Slint` and `rust`. Use `cargo build` and `cargo run` to verify your changes locally.

## Submitting Pull Requests
- Keep PRs focused on one bug or one new feature at a time to accelerate reviews.
- Please detail exactly what your PR modifies and how you verified its stability.

## Bug Reports and Feature Requests
If you encounter a bug or have a suggestion, open an Issue. Please provide:
1. A descriptive title.
2. Exact steps to reproduce the issue.
3. Relevant logs (expand the "Show Logs" button in the GUI during the run).
